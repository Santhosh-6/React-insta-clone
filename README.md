# 📸 React Instagram Clone

A simple Instagram-inspired user interface developed using React.js while learning core React concepts. This project focuses on understanding component-based architecture, props, state management, rendering dynamic content, and building responsive UI layouts.

---

## 📌 Project Overview

The React Instagram Clone was created as a practice project during the process of learning React.js fundamentals. The application recreates a simplified Instagram-style interface using reusable React components and local data.

The project demonstrates:

* Component-based development
* Dynamic rendering of data
* React props
* React state management
* Responsive UI design
* Vite project setup

## ⚙ Installation

Clone repository:

```bash
git clone https://github.com/Santhosh-6/React-insta-clone.git
```

Move into project folder:

```bash
cd React-insta-clone
```

Install dependencies:

```bash
npm install
```

---

## ▶ Running the Project

Start development server:

```bash
npm run dev
```

Open browser:

```bash
http://localhost:5173
```
